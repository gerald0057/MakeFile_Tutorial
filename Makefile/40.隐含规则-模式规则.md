# 语法
```
%.o : %.c ; <command ....>
```
其含义是，指出了怎么从所有的 ``.c`` 文件生成相应的 ``.o`` 文件的规则

# 例子
```
%.o : %.c
	$(CC) -c $(CFLAGS) $(CPPFLAGS) $< -o $@
```
把所有的 ``.c`` 文件都编译成 ``.o`` 文件

```
%.tab.c %.tab.h: %.y
	bison -d $<
```
这条规则告诉make把所有的 ``.y`` 文件都以 ``bison -d <n>.y`` 执行，然后生成 ``<n>.tab.c``
和 ``<n>.tab.h`` 文件
